JavaScript Booleans - True or False

Simple Rules:

  • Boolean = true or false (yes or no)
  • if/else = Make decisions
  • && = AND (both must be true)
  • || = OR (either can be true)
  • ? = Ternary (short if/else)

Real Life Examples:

  • Is user logged in? (true/false)
  • Is item in stock? (true/false)
  • Is form valid? (true/false)
