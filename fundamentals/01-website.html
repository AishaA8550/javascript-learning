Simple Rules:

  • Tags = <opening> content </closing>
  • Attributes = extra info like href="link"
  • Most tags need opening AND closing
  • Some tags are self-closing like <br> and <img>

Most Common HTML Tags:

  • <h1> to <h6> = Headings (big to small)
  • <p> = Paragraph
  • <a> = Link
  • <img> = Image
  • <ul> <li> = List
  • <button> = Button
  • <div> = Container box
